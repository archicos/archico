Clicknic

Clicknic is a desktop-based application built with the aim of managing data of a clinic in the form of Patient Examinations by Doctor and also Medicine Availability Data by Pharmacists.

This application was built by implementing an Object-Oriented Programming (OOP) paradigm using the Java Programming Language, and the GUI interface was implemented using Apache Netbeans. Then the Database Management System used is MySQL.

This project was developed as a requirement for passing the Object Oriented Programming and Database Courses. The development team for this project consists of 3 roles, namely Archico Sembiring as Developer, Rifqi Haikal Chairiansyah as Database Engineer, and Ella Silaban as UI/UX Design.

Sobat Listrik

The "Electricity Buddy" application is presented as an innovative solution to optimize electricity use. The Sobat Electric application development uses the user centered design method, namely an approach in designing products, systems or services that focuses attention on the end user and implements notification and warning strategies. Developers collect data by distributing questionnaires and observing similar applications to ensure the success and relevance of the application. Through this application, people can increase their knowledge about good electricity use practices, manage energy-intensive equipment, and optimize power consumption. By utilizing these applications, individuals and households can reduce energy waste, contribute to energy sustainability, and protect the environment.

RPA Talent Review Process

In the structure of a work organization, an employee, especially fresh employees, will undergo analysis and evaluation of the performance results they have developed to show the quality and value they have. Therefore, a Talent Review Process is a process for evaluating and testing the performance and quality of an employee in a company. In this case study, the talent who wants to be reviewed is talents from IDStar (IFT Group). However, due to the relatively large nominal number of employees, the ongoing process will become complex, because the filter process also requires a lot of sorting for criteria. Therefore, Robot Processing Automation (RPA) was involved to help maximize the efficiency of the time and processes required in each work flow of the Talent Review Process. With the help of robot automation, a fairly large number of employees can be processed automatically, quickly, precisely and consistently
